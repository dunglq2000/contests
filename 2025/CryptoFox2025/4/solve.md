# Задача "РАЗНОСТИ ДЛЯ FOX128"

Функция зашифрования открытого текста $\bm{a} \in V_{16}(2^8)$ имеет вид

$$g_{\bm{k}_1 \bm{k}_2}(\bm{a}) = \nu_{\bm{k}_2} h \pi' \nu_{\bm{k}_1}(\bm{a}).$$

Допустим, что мы зашифруем два открытых текста $\bm{a}$, $\bm{a}' \in V_{16}(2^8)$ с одним и тем же ключом $\bm{k}_1$, $\bm{k}_2$.

Пусть

$$\bm{a} = (a_1, a_2, \ldots, a_{16}), \quad \bm{a}' = (a_1', a_2', \ldots, a_{16}'),$$

где $a_i, a_i' \in \mathbb{F}_{2^8}$, $i = 1, 2, \ldots, 16$.

Пусть

$$\bm{k}_1 = (k_{1, 1}, k_{1, 2}, \ldots, k_{1, 16}), \quad \bm{k}_2 = (k_{2, 1}, k_{2, 2}, \ldots, k_{2, 16}).$$

После первой операции $\nu_{\bm{k}_1}$:

$$\begin{split}
    \nu_{\bm{k}_1}(\bm{a}) = (a_1 \oplus k_{1,1}, \ldots, a_{16} \oplus k_{1, 16}) = (b_1, \ldots, b_{16}), \\
    \nu_{\bm{k}_1}(\bm{a}') = (a_1' \oplus k_{1,1}, \ldots, a_{16}' \oplus k_{1, 16}) = (b_1', \ldots, b_{16}'),
\end{split}$$

где $b_i = a_i \oplus k_{1, i}$, $b_i' = a_i' \oplus k_{1, i}$ для $i = 1, \ldots, 16$.

После второй операции $\pi'$

$$\begin{split}
    \pi'(b_1, \ldots, b_{16}) = (c_1, \ldots, c_{16}), \\
    \pi'(b_1', \ldots, b_{16}') = (c_1', \ldots, c_{16}'),
\end{split}$$

где $c_i = \pi(b_i)$, $c_i' = \pi(b_i)'$.

Здесь необходимо обратить внимание. После линейной операции $h$ (умножение на матрицу $H$):

$$h(c_1, \ldots, c_{16}) = (d_1, \ldots, d_{16}), \quad h(c_1', \ldots, c_{16}') = (d_1', \ldots, d_{16}').$$

B последней операции $\nu_{\bm{k}_2}$:

$$\begin{split}
    \nu_{\bm{k}_2}(d_1, \ldots, d_{16}) = (d_1 \oplus k_{2, 1}, \ldots, d_{16} \oplus k_{2, 16}) = (e_1, \ldots, e_{16}), \\
    \nu_{\bm{k}_2}(d_1', \ldots, d_{16}') = (d_1' \oplus k_{2, 1}, \ldots, d_{16}' \oplus k_{2, 16}) = (e_1', \ldots, e_{16}').
\end{split}$$

Здесь $(e_1, \ldots, e_{16})$ и $(e_1', \ldots, e_{16}')$ являются зашифрованным текстом.

У нас есть

$$(e_1 \oplus e_1', \ldots, e_{16} \oplus e_{16}') = (e_1, \ldots, e_{16}) \oplus (e_1', \ldots, e_{16}') = (d_1 \oplus d_1', \ldots, d_{16} \oplus d_{16}'),$$

но операция $h$ линейной, поэтому

$$(d_1 \oplus d_1', \ldots, d_{16} \oplus d_{16}') = h(c_1, \ldots, c_{16}) \oplus h(c_1', \ldots, c_{16}') = h(c_1 \oplus c_1', \ldots, c_{16} \oplus c_{16}').$$

Отсюда можем сказать, что

$$(e_1 \oplus e_1', \ldots, e_{16} \oplus e_{16}') = h(c_1 \oplus c_1', \ldots, c_{16} \oplus c_{16}').$$

Матрица $H$ и её обратная матрица одинаковые. Поэтому можем писать, что

$$h(e_1 \oplus e_1', \ldots, e_{16} \oplus e_{16}') = (c_1 \oplus c_1', \ldots, c_{16} \oplus c_{16}').$$

Так как $c_i = \pi(a_i \oplus k_{1,i})$, $c_i' = \pi(a_i' \oplus k_{1,i})$, и мы можем считать $h(e_1 \oplus e_1', \ldots, e_{16} \oplus e_{16}')$, отсюда попробуем подключи $k_{1, i}$ для каждого $i = 1, 2, \ldots, 16$.
